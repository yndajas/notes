# Simple notes

Todos and URLs

## URLs

Open all

```sh
./open-urls [--recursive]
```

Open all in root

```sh
./open-urls --no-recursive
```

Open all in subfolder

```sh
./open-urls --folder path/to/folder --no-recursive
```

Open all in subfolder and descendants

```sh
./open-urls --folder path/to/folder [--recursive]
```

List, which takes the same arguments

```sh
./list-urls [--recursive]
./list-urls --no-recursive
./list-urls --folder path/to/folder --no-recursive
./list-urls --folder path/to/folder [--recursive]
```
